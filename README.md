## 📖 Overview

ShopEZ is a full-stack MERN E-Commerce application that enables users to browse products, add items to their cart, place orders, and track purchases. It also includes a powerful Admin Panel for managing products, users, and orders.

---

## ✨ Features

### 👤 User Features

* 🔐 User Registration & Login
* 🛒 Browse Products
* 📦 Product Details Page
* ➕ Add to Cart
* 💳 Checkout & Place Orders
* 📜 View Order History
* 🔍 View Order Details

### 🛠️ Admin Features

* 📊 Admin Dashboard
* 📦 Manage Products
* ➕ Add New Products
* ❌ Delete Products
* 📋 Manage Orders
* 🚚 Update Order Status
* 👥 View Registered Users

## 🚀 Installation

### 📥 Clone Repository

```bash
git clone https://github.com/prardhana34/VIP_C2-SHOPEZ-E-commerce-Application.git
```

### 🖥️ Backend Setup

```bash
cd server
npm install
npm run dev
```

### 💻 Frontend Setup

```bash
cd client
npm install
npm run dev
```
